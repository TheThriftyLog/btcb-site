# Between the Covers Bookshop

The official website for **Between the Covers Bookshop**, an independent bookshop inside the Shops at Mud Queen in Harrisburg, PA.

🌐 [betweenthecoversbookshop.com](https://betweenthecoversbookshop.com)

## About

A hand-built static site for a veteran-owned, women-owned independent bookshop. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no templates, no page builders. Designed to reflect the warm, literary character of the shop itself.

## Features

- **Custom brand design** — color palette extracted from the shop's hand-drawn logo (burgundy, gold, dusty rose)
- **Responsive layout** — mobile-first with scroll-triggered animations
- **Google Maps embed** — pinned to the shop's exact business listing
- **Square Online integration** — links directly to live inventory (1,600+ titles)
- **SVG illustrations** — hand-coded book stack with animated coffee steam
- **SEO-ready** — Open Graph tags, Twitter Cards, semantic HTML

## Tech Stack

- HTML5 / CSS3 / vanilla JS
- Google Fonts (Playfair Display, Lora, Cormorant Garamond)
- Hosted on Cloudflare Pages
- Square Online for e-commerce
- Version controlled with Git

## Local Development

```bash
cd btcb-site
python3 -m http.server 8000
# Open http://localhost:8000
```
